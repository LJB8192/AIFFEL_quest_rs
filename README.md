# AIFFEL_quest_rs

## 간단한 소개
이 repository는 저희 AIFFEL의 과제들을 수행하기 위한 git repository입니다.
아이펠 교육 과정에서 Main Quest는 5개, Exploration은 7개, GoingDeeper는 9개를 수행할 것입니다.
교육 과정동안 지속적으로 업데이트가 진행될 예정입니다.
